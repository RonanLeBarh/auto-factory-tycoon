# Auto-Factory Tycoon - V5

## Description
Auto-Factory Tycoon est un jeu de type "Idle Automation" développé en HTML5/JavaScript pur.
Le joueur gère une usine automatisée, achète des machines, collecte des ressources (Pièces et Données) et utilise des systèmes de boost (Overclock) et d'événements aléatoires pour accélérer sa progression.

## Fonctionnalités Clés
- **Double Économie** : Pièces (pour les machines) et Données (pour les boosts).
- **3 Branches de Spécialisation** : Logique, Magie, Sociale (chacune avec des bonus uniques).
- **Système Overclock (4 Niveaux)** : Boosts temporaires de production avec coûts et durées variables.
- **Quêtes** : Objectifs à compléter pour obtenir des récompenses en Données.
- **Événements Aléatoires** : Bonus ou pénalités temporaires (ex: Tempête de Données).
- **Système de Prestige (Reborn)** : Réinitialisation partielle pour un bonus permanent.
- **Sauvegarde Locale** : Progression sauvegardée automatiquement dans le navigateur.

## Comment Jouer
1. Ouvrez `index.html` dans un navigateur moderne.
2. Choisissez une branche de spécialisation.
3. Cliquez manuellement pour produire des Pièces.
4. Achetez des machines pour automatiser la production.
5. Accumulez des Données pour activer des Overclocks.
6. Complétez des quêtes pour des bonus.
7. Atteignez 10k Pièces/sec pour effectuer un Reborn et recommencer avec un bonus.

## Installation & Développement
Ce projet est conçu pour être exécuté localement ou hébergé sur un serveur statique (GitHub Pages, Netlify).
- **Prérequis** : Aucun (fonctionne avec un navigateur standard).
- **Structure** : Le code est séparé en HTML, CSS et JavaScript modulaire.
